# EQGRP Dashboard

Dashboard web pour parcourir les outils EQGRP Lost in Translation (Equation Group / NSA).

## Aperçu

- 📋 25 exploits listés avec descriptions, CVEs, cibles
- 🔧 8 utilitaires (FUZZBUNCH, SMBTOUCH, etc.)
- 🔍 Recherche et filtrage par catégorie
- 🎨 Interface sombre style "hacker"
- ⚡ Aucune dépendance Discord — standalone

## Déploiement Railway

1. Créer un service depuis ce repo
2. Pas de variables d'environnement requises
3. Generate Domain dans Settings → Networking

## Développement local

```bash
npm install
npm start
```

→ http://localhost:3000
